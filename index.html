Penggunaan untuk Masyarakat Umum

Memilih Pengguna

  1. Buka aplikasi Si Gadis Tanjung.
  2. Aplikasi akan menampilkan tiga pilihan pengguna.
  3. Tekan tombol Tamu.

  4. Anda akan melanjutkan penggunaan aplikasi sebagai pengguna Masyarakat Umum.

Memilih Bahasa

  1. Setelah memilih tipe pengguna, aplikasi akan menampilkan pilian bahasa.
  2. Aplikasi akan menampilkan tiga pilihan bahasa: Bahasa Banjar, Bahasa Dayak, Bahasa Indonesia.
  3. Tekan tombol bahasa yang anda inginkan.

  4. Anda akan melanjutkan penggunaan aplikasi dengan bahasa yang anda pilih.

Mengganti Bahasa

  1. Tekan tombol yang bergambar bola dunia pada pojok kanan atas aplikasi.
  2. Aplikasi akan menampilkan tiga pilihan bahasa: Bahasa Banjar, Bahasa Dayak, Bahasa Indonesia.
  3. Tekan tombol bahasa yang anda inginkan.

  4. Anda akan melanjutkan penggunaan aplikasi dengan bahasa yang anda pilih.

Membuka Fitur KIE Artikel dan Resep

  1. Pada tampilan layar utama aplikasi, geser layar aplikasi.
  2. Pilih konten artikel atau resep yang anda akan baca.
  3. Aplikasi akan menampilkan artikel atau resep yang anda pilih.

  4. Khusus untuk artikel, aplikasi juga akan memutar suara dub. Tekan tombol Play atau Pause pada pojok kanan bawah aplikasi untuk melanjutkan memutar atau menghentikan suara.

Membuka Fitur KIE Video Eksternal

  1. Pada tampilan layar utama aplikasi, geser layar aplikasi.
  2. Pilih thumbnail video yang akan anda tonton.

  3. Aplikasi akan membuka peramban atau aplikasi eksternal untuk memutar video.

Memeriksa Statistik

  1. Pilih tombol Statistik pada halaman utama aplikasi.

  2. Aplikasi akan menampilkan prevalensi statistik tiap kecamatan

Menghitung Status Gizi Anak (Kalkulator Gizi)

  1. Pilih tombol Kalkulator Gizi pada halaman utama aplikasi.
  2. Isi formulir informasi umum anak yang meliputi Nama Anak, Jenis Kelamin dan Tanggal Lahir, lalu tekan tombol Berikutnya.
  3. Isi formulir pengukuran anak yang meliputi Tanggal Ukur,Berat Badan, Panjang/Tinggi Anak. Tekan kotak centang Diukur hari ini jika pengukuran dilakukan pada hari yang sama. Lalu tekan tombol Periksa Status Gizi.

  4. Aplikasi akan menampilkan hasil perhitungan status gizi anak, dekripsi dan saran.

Memeriksa Status Perkembangan Anak (SDIDTK)

  1. Pilih tombol SDIDTK pada halaman utama aplikasi.
  2. Pilih umur anak yang sesuai pada tombol Umur xx - xx bulan
  3. Jawab pertanyaan kuesioner dengan menekan tombol Ya atau Tidak sesuai keadaan anak. Jawab pertanyaan kuesioner sampai selesai.

  4. Jika kuesioner telah selesai, aplikasi akan menampilkan status perkembangan anak dan saran.

Berganti Pengguna

  1. Tekan tombol yang bergambar Avatar dalam Lingkaran pada pojok kanan atas aplikasi.
  2. Aplikasi akan menampilkan tiga pilihan pengguna.
  3. Tekan tombol tipe pengguna yang sesuai dengan pengguna baru.

  4. Anda akan melanjutkan penggunaan aplikasi dengan tipe pengguna yang anda pilih.

Penggunaan untuk Petugas Kesehatan

Masuk Sebagai Petugas

  1. Pada halaman pemilihan pengguna, tekan tombol Petugas.
  2. Masukkan detail kredensial Email dan Kata Sandi, dan pilih Kecamatan yang sesuai.
  3. Tekan tombol Masuk.

  4. Aplikasi akan melanjutkan penggunaan aplikasi sebagai pengguna Petugas Kesehatan.

Memverifikasi Data Perhitungan yang Masuk

  1. Pilih tombol Pendataan pada halaman utama aplikasi.
  2. Tekan Data Masuk.

  3. Pilih salah satu data perhitungan dari Kalkulator Gizi yang masuk dari pengguna Orang Tua.

  4. Tekan tombol Verifikasi untuk memverifikasi data yang masuk ke data balita atau tombol Tolak/Arsip untuk menolak hasil perhitungan dan mengarsipkan data tersebut.

  5. Tekan tombol Edit jika ingin mengubah data sebelum memverifikasi.

  6. Edit kolom data yang diperlukan, tekan tombol Ubah jika telah selesai. Selanjutnya tekan tombol Verifikasi untuk memverifikasi data yang telah berubah.

Menambah Data Balita

  1. Pilih tombol Pendataan pada halaman utama aplikasi.

  2. Tekan Data Balita.

  3. Tekan Tambah Data Balita.

  4. Isi formulir penambahan data balita yang baru.

  5. Tekan tombol Tambahkan Balita untuk menampilkan hasil perhitungan status gizi.

  6. Aplikasi akan menampilkan data dan hasil perhitungan status gizi balita yang telah di input.

  7. Tekan tombol Simpan untuk menyimpan data balita yang baru.

Mengelola Data Balita

  1. Pilih tombol Pendataan pada halaman utama aplikasi.

  2. Tekan Data Balita.

  3. Tekan kolom Cari Balita Berdasarkan Nama..., dan ketik nama balita yang ingin dicari.

  4. Pilih salah satu balita dari daftar balita yang ditampilkan.

  5. Untuk mengedit, tekan tombol Pensil pada pojok kanan atas aplikasi.

  6. Edit data balita pada formulir yang ditampilkan, tekan tombol Ubah jika ingin menyimpan hasil perubahan data.

  7. Untuk menghapus, tekan tombol Tong Sampah pada pojok kanan atas aplikasi.

  8. Tekan Ya untuk mengkonfirmasi penghapusan data balita.

Menambah Data Pengukuran Balita

  1. Pilih salah satu balita yang ingin ditambahkan pengukurannya.

  2. Tekan tombol Tambah Pengukuran.

  3. Isi formulir pengukuran dan tekan tombol Periksa Status Gizi untuk menghitung data pengukuran.

  4. Aplikasi akan menampilkan hasil perhitungan pengukuran. Tekan tombol Simpan untuk menyimpan pengukuran.

Mengeola Data Pengukuran Balita

  1. Pilih salah satu balita yang ingin ditambahkan pengukurannya.

  2. Pilih pengukuran yang ditampilkan.

  3. Untuk mengedit, tekan tombol Edit.

  4. Edit data pengukuran yang ditampilkan, tekan tombol Simpan untuk menyimpan hasil pengukuran.

  5. Untuk menghapus, tekan tombol Hapus.

  6. Tekan tombol Ya untuk mengkonfirmasi penghapusan data pengukuran.

Mengekspor Data

  1. Pilih tombol Pendataan pada halaman utama aplikasi.

  2. Tekan tombol Ekspor Data.

  3. Pilih tombol Ekspor Data Balita untuk mengekspor data balita.

  4. Pilih tombol Ekspor Data Pengukuran Balita untuk mengekspor data pengukuran balita.

  5. Aplikasi akan mengunduh berkas xlsx yang kompatibel dengan e-PPBGM.
